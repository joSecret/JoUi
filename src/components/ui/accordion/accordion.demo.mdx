# Accordion

Collapsible panels built on the native `<details>`/`<summary>` elements. No JavaScript. Full keyboard and screen reader support out of the box.

## Installation

```bash
npx joui add accordion --platform astro
```

```bash
npx joui add accordion --platform drupal
```

## Usage

```astro live
---
import Accordion from '@ui/accordion/Accordion.astro'
---
<Accordion items={[
  { title: 'What is joui?', content: '<p>A multi-platform CSS component library.</p>' },
  { title: 'Is JS required?', content: '<p>No — built on native details.</p>', open: true },
]} />
```
