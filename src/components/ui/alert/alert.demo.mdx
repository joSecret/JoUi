# Alert

Contextual feedback messages for typical user actions. Choose from color variants to convey meaning or use the dismissible variant for temporary notices.

## Installation

```bash
npx joui add alert --platform astro
```

```bash
npx joui add alert --platform drupal
```

## Usage

```astro live
---
import Alert from '@ui/alert/Alert.astro'
---
<Alert color="primary">A simple primary alert — check it out!</Alert>
```
