# Breadcrumb

Navigation aid that shows the user's current location within the site hierarchy. Built on a semantic `<ol>` wrapped in a `<nav>`. No JavaScript required.

## Installation

```bash
npx joui add breadcrumb --platform astro
```

```bash
npx joui add breadcrumb --platform drupal
```

## Usage

```astro live
---
import Breadcrumb from '@ui/breadcrumb/Breadcrumb.astro'
---
<Breadcrumb items={[
  { label: 'Home', url: '/' },
  { label: 'Library', url: '/library' },
  { label: 'Data', active: true },
]} />
```
