# Button Group

Groups multiple buttons into a visually connected row. Removes inner border radii and collapses borders. No JavaScript required.

## Installation

```bash
npx joui add button-group --platform astro
```

```bash
npx joui add button-group --platform drupal
```

## Usage

```astro live
---
import ButtonGroup from '@ui/button-group/ButtonGroup.astro'
import Button from '@ui/button/Button.astro'
---
<ButtonGroup label="Example group">
  <Button>Left</Button>
  <Button>Middle</Button>
  <Button>Right</Button>
</ButtonGroup>
```
