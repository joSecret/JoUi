# Button

Flexible button component supporting multiple colors, sizes, styles, and states. Can render as `button`, `a`, `span`, `input`, or `summary`.

## Installation

```bash
npx joui add button --platform astro
```

```bash
npx joui add button --platform drupal
```

## Usage

```astro live
---
import Button from '@ui/button/Button.astro'
---
<Button label="Click me" />
```
