# Card

A flexible content container. Compose the internals using `.card-header`, `.card-body`, `.card-footer`, `.card-title`, `.card-text`, and image helpers. No JavaScript required.

## Installation

```bash
npx joui add card --platform astro
```

```bash
npx joui add card --platform drupal
```

## Usage

```astro live
---
import Card from '@ui/card/Card.astro'
---
<Card>
  <img src="https://picsum.photos/400/200" class="card-img-top" alt="Sample image">
  <div class="card-body">
    <h5 class="card-title">Card title</h5>
    <p class="card-text">Some quick example text to build on the card title.</p>
    <a href="#" class="button button-primary">Go somewhere</a>
  </div>
</Card>
```
