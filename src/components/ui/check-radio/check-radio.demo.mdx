# Checks & Radios

Custom-styled checkbox, radio button, and toggle switch controls. All styled via CSS with no browser defaults. No JavaScript required.

## Installation

```bash
npx joui add check-radio --platform astro
```

```bash
npx joui add check-radio --platform drupal
```

## Usage

```astro live
---
import CheckRadio from '@ui/check-radio/CheckRadio.astro'
---
<CheckRadio id="default-check" label="Default checkbox" />
```

## Examples

### Checkboxes

<CheckRadio id="check-1" label="Default checkbox" />
<CheckRadio id="check-2" label="Checked checkbox" checked />
<CheckRadio id="check-3" label="Disabled checkbox" disabled />
