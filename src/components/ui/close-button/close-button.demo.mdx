# Close Button

A reusable × dismiss button. Used inside alerts, modals, toasts, and offcanvas panels. The button itself is presentational — the parent component's JS handles the dismiss action. No JavaScript in the button.

## Installation

```bash
npx joui add close-button --platform astro
```

```bash
npx joui add close-button --platform drupal
```

## Usage

```astro live
---
import CloseButton from '@ui/close-button/CloseButton.astro'
---
<CloseButton />
```
