# Dropdown

Toggleable menu that opens on button click. Closes on outside click, Escape key, and automatically positions the menu. Requires `dropdown.js`.

## Installation

```bash
npx joui add dropdown --platform astro
```

```bash
npx joui add dropdown --platform drupal
```

## Usage

```astro live
---
import Dropdown from '@ui/dropdown/Dropdown.astro'
---
<Dropdown
  label="Options"
  color="secondary"
  items={[
    { label: 'Action', href: '#' },
    { label: 'Another action', href: '#' },
    { divider: true },
    { label: 'Something else', href: '#' },
  ]}
/>
```
