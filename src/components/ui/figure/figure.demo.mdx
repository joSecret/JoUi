# Figure

Semantic `<figure>` element with an optional `<figcaption>`. Supports caption alignment and rounded image corners. No JavaScript required.

## Installation

```bash
npx joui add figure --platform astro
```

```bash
npx joui add figure --platform drupal
```

## Usage

```astro live
---
import Figure from '@ui/figure/Figure.astro'
---
<Figure
  src="https://picsum.photos/400/250"
  alt="Sample image"
  caption="A sample caption below the image."
/>
```

## Examples

### With caption

<Figure
  src="https://picsum.photos/400/250"
  alt="Sample landscape"
  caption="Fig. 1 — A sample landscape image."
/>

```astro
<Figure
  src="https://picsum.photos/400/250"
  alt="Sample landscape"
  caption="Fig. 1 — A sample landscape image."
/>
```

### Caption alignment

<Figure src="https://picsum.photos/400/200?1" alt="Photo" caption="Centered caption" captionAlign="center" />
<Figure src="https://picsum.photos/400/200?2" alt="Photo" caption="End-aligned caption" captionAlign="end" />

```astro
<Figure src="..." alt="Photo" caption="Centered caption" captionAlign="center" />
<Figure src="..." alt="Photo" caption="End-aligned caption" captionAlign="end" />
```

### Rounded corners

<Figure src="https://picsum.photos/200/200?3" alt="Photo" caption="Rounded" rounded />
<Figure src="https://picsum.photos/200/200?4" alt="Avatar" caption="Circle" rounded="circle" />

```astro
<Figure src="..." alt="Photo" rounded />
<Figure src="..." alt="Avatar" rounded="circle" />
```
