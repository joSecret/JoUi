# Floating Label

Form controls where the label animates from inside the input to above it on focus or when filled. Pure CSS using `:placeholder-shown`. Works with `input`, `select`, and `textarea`. No JavaScript required.

## Installation

```bash
npx joui add floating-label --platform astro
```

```bash
npx joui add floating-label --platform drupal
```

## Usage

```astro live
---
import FloatingLabel from '@ui/floating-label/FloatingLabel.astro'
---
<FloatingLabel id="demo-email" type="email" label="Email address" />
```
