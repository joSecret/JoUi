# Image

Responsive image component with optional thumbnail frame and rounded corner utility classes. Renders a semantic `<img>` element. No JavaScript required.

## Installation

```bash
npx joui add image --platform astro
```

```bash
npx joui add image --platform drupal
```

## Usage

```astro live
---
import Image from '@ui/image/Image.astro'
---
<Image src="https://picsum.photos/400/250" alt="Sample photo" />
```
