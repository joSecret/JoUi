# Nav

Flexible navigation links. Supports plain nav, tabs, and pills variants. Tabs with panels require the included `nav.js` behavior.

## Installation

```bash
npx joui add nav --platform astro
```

```bash
npx joui add nav --platform drupal
```

## Usage

```astro live
---
import Nav from '@ui/nav/Nav.astro'
---
<Nav items={[
  { label: 'Active', href: '#', active: true },
  { label: 'Link', href: '#' },
  { label: 'Another link', href: '#' },
  { label: 'Disabled', disabled: true },
]} />
```

### Tabs with panels

  { label: 'Tab 1', tabTarget: 'demo-panel1', active: true },
  { label: 'Tab 2', tabTarget: 'demo-panel2' },
  { label: 'Tab 3', tabTarget: 'demo-panel3' },
]} />
<div class="tab-content">
  <div class="tab-pane active" id="demo-panel1" role="tabpanel">
    <p>Content for Tab 1. Click the tabs above to switch panels.</p>
  </div>
  <div class="tab-pane" id="demo-panel2" role="tabpanel">
    <p>Content for Tab 2.</p>
  </div>
  <div class="tab-pane" id="demo-panel3" role="tabpanel">
    <p>Content for Tab 3.</p>
  </div>
</div>

```astro

  <div class="tab-pane active" id="panel1" role="tabpanel">Content 1</div>
  <div class="tab-pane" id="panel2" role="tabpanel">Content 2</div>
```
