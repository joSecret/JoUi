# Navbar

Responsive navigation bar. Collapses into a hamburger menu on mobile. Requires `navbar.js` for the toggle behavior.

## Installation

```bash
npx joui add navbar --platform astro
```

```bash
npx joui add navbar --platform drupal
```

## Usage

```astro live
---
import Navbar from '@ui/navbar/Navbar.astro'
---
<Navbar
  brand="My Site"
  brandUrl="/"
  items={[
    { label: 'Home', href: '/', active: true },
    { label: 'About', href: '/about' },
    { label: 'Contact', href: '/contact' },
  ]}
/>
```
