# Pager

Navigation component for paginated content. Displays first, previous, page numbers, ellipsis, next, and last links.

## Installation

```bash
npx joui add pager --platform astro
```

```bash
npx joui add pager --platform drupal
```

## Usage

```astro live
---
import Pager from '@ui/pager/Pager.astro'
---
<Pager
  items={{
    previous: { href: '#' },
    pages: { 1: { href: '#' }, 2: { href: '#' }, 3: { href: '#' } },
    next: { href: '#' },
  }}
  current={2}
/>
```

## Examples

### Default

<Pager items={pages} current={3} />

```astro
<Pager items={pages} current={3} />
```

### With first/last + ellipsis

<Pager items={fullPages} current={5} ellipses={{ previous: true, next: true }} />
