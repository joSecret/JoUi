# Placeholder

Skeleton loading indicators used while content is loading. Two animation styles: `glow` (opacity pulse) and `wave` (shimmer). No JavaScript required.

## Installation

```bash
npx joui add placeholder --platform astro
```

```bash
npx joui add placeholder --platform drupal
```

## Usage

```astro live
---
import Placeholder from '@ui/placeholder/Placeholder.astro'
---
<Placeholder lines={[{ cols: 7 }, { cols: 4 }, { cols: 12 }]} />
```

## Examples

### Glow (default)

<Placeholder lines={[{ cols: 7 }, { cols: 4 }, { cols: 12 }, { cols: 8 }]} />

<Placeholder lines={[{ cols: 7 }, { cols: 4 }, { cols: 12 }, { cols: 8 }]} />
