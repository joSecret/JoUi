# Progress

Horizontal progress bar with color, striped, and animated stripe variants. Values are clamped to 0–100 automatically.

## Installation

```bash
npx joui add progress --platform astro
```

```bash
npx joui add progress --platform drupal
```

## Usage

```astro live
---
import Progress from '@ui/progress/Progress.astro'
---
<Progress value={60} color="primary" label={true} />
```
