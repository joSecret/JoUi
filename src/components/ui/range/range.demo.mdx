# Range

Custom-styled range slider (`<input type="range">`). Consistent appearance across Chrome, Firefox, and Safari. No JavaScript required.

## Installation

```bash
npx joui add range --platform astro
```

```bash
npx joui add range --platform drupal
```

## Usage

```astro live
---
import Range from '@ui/range/Range.astro'
---
<Range id="example" label="Example range" />
```
