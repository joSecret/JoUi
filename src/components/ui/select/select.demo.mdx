# Select

Dropdown select. Overrides Drupal's `select.html.twig`.

## Usage

```astro live
---
import Select from '@ui/select/Select.astro'
---
<Select emptyOption="Choose..." options={[
  { value: '1', label: 'Option 1' },
  { value: '2', label: 'Option 2' },
]} />
```
