# Spinner

Animated loading indicator. Two styles: `border` (rotating ring) and `grow` (pulsing circle). Fully accessible with a visually hidden label.

## Installation

```bash
npx joui add spinner --platform astro
```

```bash
npx joui add spinner --platform drupal
```

## Usage

```astro live
---
import Spinner from '@ui/spinner/Spinner.astro'
---
<Spinner color="primary" />
```
