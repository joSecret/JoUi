# Table

CSS-styled table. Overrides Drupal's `table.html.twig`. No sorting or filtering — styling only.

## Installation

```bash
npx joui add table --platform astro
```

```bash
npx joui add table --platform drupal
```

## Usage

```astro live
---
import Table from '@ui/table/Table.astro'
---
<Table striped hover>
  <thead>
    <tr><th>Name</th><th>Role</th><th>Status</th></tr>
  </thead>
  <tbody>
    <tr><td>Alice</td><td>Admin</td><td>Active</td></tr>
    <tr><td>Bob</td><td>Editor</td><td>Active</td></tr>
    <tr><td>Carol</td><td>Viewer</td><td>Inactive</td></tr>
  </tbody>
</Table>
```
